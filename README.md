# Hello Encryptor

Hello Encryptor is a secure password management application developed by [Helloware](https://helloware.tech). It allows you to encrypt and manage your passwords in a secure way. You can encrypt passwords by user, export them, and import them as needed.

## Features

- Encrypt passwords by user for added security.
- Export encrypted passwords to a secure file.
- Import encrypted passwords to restore your data.
- User-friendly interface for easy password management.

## Install

Clone the repo and install dependencies:

```bash
git clone --depth 1 https://github.com/robertocarlero/helloencryptor.git your-project-name
cd your-project-name
npm install
```

**Having issues installing? See our [debugging guide](https://github.com/electron-react-boilerplate/electron-react-boilerplate/issues/400)**

## Starting Development

Start the app in the `dev` environment:

```bash
npm start
```
